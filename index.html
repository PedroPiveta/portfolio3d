Meus Projetos

Keep Inventory

Aplicativo Flutter desenvolvido para facilitar a gestão de inventário de reagentes em laboratórios. Com ele, é possível registrar, organizar e monitorar o estoque de reagentes, incluindo informações como validade, quantidade disponível e localização. O app oferece busca rápida, notificações para reagentes próximos do vencimento e relatórios detalhados para otimizar a reposição e evitar desperdícios. Ideal para laboratórios que buscam eficiência e precisão no gerenciamento de seus insumos.

Tecnologias: Flutter e Spring Boot

Telegram Bot

Esse bot se trata de um sistema integrado com Telegram e painel administrativo que permite gerenciar pedidos de forma dinâmica e eficiente. Com comunicação em tempo real via WebSockets, oferece funcionalidades como realização de pedidos, notificações automáticas, gerenciamento de cardápio e acompanhamento do status do pedido, garantindo uma experiência rápida e intuitiva para clientes e restaurantes.

Tecnologias: Python, Flask e Websockets

Agenda Api

A Agenda API é uma solução simples e eficiente para gerenciar compromissos. Ela permite criar, atualizar, listar e excluir eventos com facilidade. Ideal para integrar em sistemas que precisam de funcionalidades básicas de agenda, oferecendo endpoints claros e estrutura leve para rápida implementação.

Tecnologias: Csharp e Dotnet
